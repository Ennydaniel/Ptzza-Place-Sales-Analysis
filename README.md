🍕 Pizza Place Sales Analysis


📊 Project Overview


This project analyzes sales data from a pizza restaurant to uncover customer ordering patterns, identify best-selling products, and generate revenue-driven insights that can support business decision-making.



🎯 Objectives




Identify top-selling pizza types and sizes


Analyze peak order times and customer demand trends


Evaluate revenue performance across different categories


Provide data-driven recommendations for improving sales





🛠️ Tools & Technologies




Python


Jupyter Notebook


Pandas


Matplotlib / Seaborn





📁 Dataset


The dataset contains transaction-level sales data, including:




Order ID


Date & Time of purchase


Pizza type and size


Quantity sold


Price





🔍 Key Analysis Performed




Data cleaning and preprocessing


Exploratory Data Analysis (EDA)


Sales trend analysis (daily, hourly)


Product performance evaluation


Revenue breakdown





📈 Key Insights




Certain pizza types consistently outperform others in sales


Peak sales occur during specific hours (e.g., lunch/dinner periods)


Larger-sized pizzas contribute more to overall revenue


Seasonal or weekly trends influence customer demand





💡 Recommendations




Increase stock/preparation during peak hours


Promote high-performing pizza categories


Introduce targeted discounts during low-demand periods


Optimize menu offerings based on customer preferences
